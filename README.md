# Blocking
Cocos2D Puzzle Game

## 프로젝트
* 제작기간 : 26시간
* Visual Studio 20
* Cocos2d 3.4

반사를 이용한 퍼즐게임입니다. 플레이어는 장애물을 설치하고 회전시킬 수 있습니다. 공이 장애물에 부딪쳐 이동하여 목적지에 도달하면 됩니다.
